矩阵初等变换：

- 交换矩阵的两行（列）；
- 以非 0 常数乘矩阵的某一行（列）；
- 把某一行的常数倍加到另一行（列）上。

矩阵的初等行变换不会改变矩阵所代表的线性方程组的解。

$n$ 阶矩阵 $A$ 的 $(i, j)$ 余子式记为 $M_{ij}$，表示将该矩阵的第 $i$ 行和第 $j$ 列移除以后得到的 $n-1$ 阶矩阵。
